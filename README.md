# NIS Company Website

A modern, responsive website for Ninja Informatics Solutions built with Next.js, TypeScript, and shadcn/ui components.

## 🚀 Features

- **Modern Design**: Professional company landing page with custom purple theme
- **Responsive Layout**: Mobile-friendly navigation with hamburger menu
- **Component Architecture**: Well-organized components split into logical sections
- **shadcn/ui Integration**: Using latest shadcn/ui v4 components
- **Custom Theme**: Purple color scheme matching the provided theme configuration

## 📁 Project Structure

```
src/
├── app/
│   ├── globals.css          # Global styles with custom theme
│   ├── layout.tsx           # Root layout
│   └── page.tsx             # Main page component
├── components/
│   ├── layout/
│   │   ├── header.tsx       # Navigation header
│   │   └── footer.tsx       # Footer component
│   ├── pages/
│   │   ├── home-page.tsx    # Homepage content
│   │   ├── cd4-lims-page.tsx # CD4 LIMS product page
│   │   ├── about-us-page.tsx # About us page
│   │   └── contact-us-page.tsx # Contact information
│   └── ui/                  # shadcn/ui components
│       ├── button.tsx
│       ├── card.tsx
│       ├── badge.tsx
│       ├── navigation-menu.tsx
│       └── sheet.tsx
└── lib/
    └── utils.ts             # Utility functions
```

## 🛠 Technology Stack

- **Next.js 14**: React framework with App Router
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first CSS framework
- **shadcn/ui**: Modern UI component library
- **Lucide React**: Beautiful icon set

## 📱 Pages

1. **Home**: Company and product overview with features
2. **CD4 LIMS**: Detailed product information and capabilities
3. **About Us**: Company mission, vision, and values
4. **Contact Us**: Contact information and form

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
